# mindful-app

## 5/28  
以下の様なアプリをReact Native を使って実装。  
1. アプリを開いたら、音声一覧画面が表示されている。  
2. 音声をタップしたら、以下の物が表示されている画面に遷移して、音声(MP3ファイル形式)を再生する  
  - 音声に紐付いている画像  
  - 停止ボタン  
  - シークバー  
  - 再生時間  
  - 前の画面に戻るボタン  
3. 停止ボタンを押したら音声を停止して、停止ボタンを再生ボタンに変える。  
4. 前の画面に戻るボタンを押したら一覧画面にもどる
